### 基础的linux命令
clear：清除屏幕 --或操作Ctrl+l
echo 'xxx'：往控制台输出信息
echo 'xxx' > xxx.txt：输出信息到新的生成文件
ll：查看当前目录下的非隐藏内容
find ./：查看指定目录下的全部内容
find ./ -type f：只看文件
rm xxx：删除文件
mv xxx xxx：重命名
cat xxx：查看指定文件的内容
vim xxx：编辑文件内容
  i 开启编辑
  "esc" + ":" + "set num" 显示行号
  "esc" + ":" + "wq"      保存并退出
  "esc" + ":" + "q!"      强制退出
